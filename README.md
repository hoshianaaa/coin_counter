coin counter
=================

## About

硬貨を認識するアプリケーション  
カメラまたは写真上の硬貨を認識し合計金額を計算する  
  
  coin_detection_from_camera.ipynb : カメラ映像から硬貨認識  
    
  ・実験環境  
  写真のように赤い紙の上に硬貨を置く  
  cameraを20cmほど上に設置する
